Usage
=====

Install in editable mode (recommended during development)::

    pip install -e .[docs]

Run the app from anywhere::

    multiformat-app

Notes & prerequisites
---------------------

- FFmpeg must be installed and on your PATH.
- ``pdf2image`` requires Poppler to be installed and on your PATH.
- ``pypandoc`` requires Pandoc installed.
- ``docx2pdf`` uses Microsoft Word on Windows to convert DOCX to PDF.
